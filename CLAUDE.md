# CLAUDE.md

このファイルは Claude Code がこのリポジトリで作業する際のガイドラインです。

## プロジェクト概要

- **todo**（ToDo 管理）プロジェクト。現在は初期段階。
- 今後、機能や構成が決まり次第このセクションを更新する。

## 言語・コミュニケーション規約

- このプロジェクトでのやり取り・回答は **日本語** で行うこと。

## 会話履歴・証跡の管理ルール

作業の証跡を残すため、会話の履歴と概要を Markdown で管理し、git にコミットする。

- **保存先**: `docs/history/`
- **ファイル名**: `YYYY-MM-DD-<topic>.md`
  （例: `2026-06-17-claude-md-setup.md`）
- **各エントリに記録する項目**:
  - 日付
  - 依頼内容（ユーザーの要望）
  - 対応概要（実施した作業）
  - 決定事項
  - 成果物（変更・新規作成したファイルなど）
- セッションの区切りごとに履歴ファイルを作成・更新し、git にコミットする。

### 履歴ファイルのテンプレート

```markdown
# YYYY-MM-DD <タイトル>

## 日付
YYYY-MM-DD

## 依頼内容
- （ユーザーからの要望を記載）

## 対応概要
- （実施した作業を記載）

## 決定事項
- （会話の中で決めたことを記載）

## 成果物
- （変更・新規作成したファイルなどを記載）
```
