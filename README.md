# TODO

ローカルファースト ＋ Bring Your Own Storage の TODO 管理 PWA。

オフラインでも使えて、スマホにもパソコンにもアプリとして入れられます。データはあなたが選んだクラウド（Dropbox / Google Drive）に保存され、複数の端末で同じ TODO を使えます。

> このページは「使い方の説明書（ユーザーマニュアル）」です。開発者向けの情報は末尾にまとめています。

---

## このアプリでできること

- **TODO の管理**: タスクの追加・編集・完了・削除。期限・優先度・メモ・タグも付けられます。
- **オフラインで使える**: ネットがなくても読み書きできます。
- **アプリとして使える**: ホーム画面やアプリ一覧に追加して、ふつうのアプリのように開けます。
- **複数の端末で共有**: クラウド（Dropbox / Google Drive）と連携すると、別の端末でも同じ TODO を使えます。
- **データはあなたのもの**: データはあなたが選んだクラウドに保存されます（提供者のサーバには送られません）。

---

## はじめかた

1. ブラウザでアプリの URL を開きます。連携しなくても、その端末だけで TODO はすぐ使えます。
2. 複数端末で使いたい場合は、あとから「クラウド連携」を設定します（下記参照）。

### アプリとしてインストールする

「設定」の「アプリ」に、お使いの環境に合った案内が表示されます。

- **パソコン（対応ブラウザ）**: ブラウザのメニューからインストールできます。アプリ内に「インストール」ボタンが出ることもあります。
- **iPhone / iPad（Safari）**: 共有メニューから「ホーム画面に追加」を選びます。
- **Android など**: ブラウザのメニューからインストールできます。

---

## 基本の使い方

画面下部（スマホ）／左側（パソコン）のナビゲーションで切り替えます。

- **タスク**: TODO の一覧。ここで追加・編集・完了・削除をします。
- **設定**: 連携・同期・データ・アプリの設定。

タスクには次を設定できます。

- **期限** / **優先度** / **メモ** / **タグ**

---

## 複数の端末で使う（クラウド連携）

「設定」→「クラウド連携」から行います。

1. **「Dropbox と連携」** または **「Google Drive と連携」** を押し、画面の案内に従ってログイン・許可します。
2. 連携すると同じデータを複数端末で使えます。
3. 連携をやめたいときは **「連携を解除」**。再ログインが必要になったときは **「再連携」** が表示されます。

### 同期設定（連携済みのときに表示）

「設定」→「同期設定」で同期のしかたを選べます。

- **手動のみ**: 自分で同期するときだけ同期します。
- **自動（間隔）**: 一定の **同期間隔（分）** ごとに自動で同期します。
- **今すぐ同期**: その場で同期します。

---

## 競合（「要解決」）が出たときの直し方

別の端末で同じタスクを別々に編集すると、タスク一覧に **「要解決」** のバッジが付くことがあります。その場合は次の手順で直します。

1. 一覧の「要解決」のタスクから **「同期エラーを解決」** 画面を開きます。
2. 項目ごとに **「オンライン」**（クラウド側の値）か **「保留中」**（まだ反映していない自分の変更）を選びます。**「直接入力」** で自分で書き直すこともできます。
3. 片方が削除だった場合は **「編集版を残す」** か **「削除を適用」** を選びます。
4. **「プレビュー」** で結果を確認し、**「確定する」** で確定します。

## データの管理（バックアップ・取り込み・削除）

「設定」→「データ」から行います。

### バックアップ（書き出し）

- **タスクをバックアップ**: タスクを丸ごと保存（あとでこのアプリに読み込んで元に戻せます）。
- **タスク (Markdown)** / **タスク (CSV)**: 他のアプリや表計算で読むための書き出し（読み戻し用ではありません）。
- **設定をバックアップ** / **全体をバックアップ（タスク＋設定）**。

### 取り込み（読み込み）

- **バックアップから読み込む**: 保存したバックアップファイルを選んで取り込みます。
  - タスクは「取り込み」＝**同じタスクは新しい方を採用し、別のタスクは両方残します**（上書きで消えることはありません）。
  - 設定を含むファイルは、確認のうえこの端末の設定を上書きします。
  - 取り込み前に内容を確認できます。

### ローカルデータ（表示がおかしい・同期が直らないとき）

「データ」の下部にあります。**クラウド上のデータは消えません**。

- **ローカルデータを削除**: この端末に保存したデータだけを削除します。
- **クラウドから復元**: この端末のデータを削除し、クラウドから取り直して作り直します（同期の不調を直したいとき）。オンラインで実行してください。
- **連携を解除してすべて削除**: 連携を解除し、この端末のデータと設定をすべて削除して初期状態に戻します。

---

## 困ったときは

- **表示がおかしい / 同期が直らない**: 「設定」→「データ」→ **「クラウドから復元」** を試してください（クラウド上のデータは消えません）。
- **それでも直らない**: アプリをいったん閉じて開き直す、または再インストールしてください。
- **不具合の報告**: アプリ内の報告導線、または本リポジトリの Issue からお知らせください。

---

## プライバシー

- データはあなたの端末と、あなたが連携したクラウド（Dropbox / Google Drive）に保存されます。**アプリ提供者のサーバには送られません**（Bring Your Own Storage）。
- 連携には最小限の権限のみを使います（アプリ専用フォルダ等）。

npm run dev      # 開発サーバ
